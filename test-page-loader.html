Page Loader Test

Test Controls

Instructions

  1. Click "Show Loader" to display the page loader
  2. Click "Hide Loader" to hide it normally
  3. Click "Force Hide Loader" to force hide it
  4. Click "Debug Loader" to see debug information in console
  5. Click "Simulate Back Navigation" to test back navigation handling
  6. Use browser back/forward buttons to test real navigation

Expected Behavior

  • Loader should show when navigating to this page
  • Loader should hide when page is fully loaded
  • Loader should show briefly when using browser back/forward
  • Loader should not get stuck in loading state
  • Force hide should work even if loader is stuck
